# Horta Inteligente Sustentável

Projeto de pesquisa desenvolvido pelo PET Elétrica da UFMT.

## Objetivo

Desenvolver um sistema inteligente de irrigação automatizada para uma horta escolar utilizando sensores ambientais, ESP32, APi`s, MQTT e monitoramento remoto.

## Sensores Utilizados

 DHT22 (temperatura e umidade do ar)
 Sensor capacitivo de umidade do solo

## Atuadores

 
 Válvula solenoide  bomba d'água
 Bomba d`agua

## Controlador

 ESP32

## Funcionalidades

 Monitoramento da umidade do solo
 Monitoramento da temperatura do ar
 Irrigação automática
 Interface Web local
 Futuras integrações com MQTT e API climática
 Calculo VDP

## Status
